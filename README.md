# MIME Messenger

Полнофункциональный мессенджер, похожий на Telegram, с поддержкой:
- Личных и групповых чатов
- Голосовых и видеосообщений
- Видеозвонков с демонстрацией экрана
- Сквозного шифрования (E2EE)
- Push-уведомлений
- 2FA
- API для ботов
- Монетизации через Stripe
- Админ-панели

## Запуск в production

1. Скопируйте `.env.example` в `.env` и заполните переменные.
2. Установите Docker и Docker Compose.
3. Выполните `docker-compose up -d`.
4. Откройте `https://ваш-домен`.
